It's all about JavaScript

Jan 6th, 2014

In a world of single page web applications, Cordova, node.js, x86 virtualization in JavaScript and Espruino it appears the world of web and mobile apps, the world of microprocessors and subsequently the real world rely more and more on a technology that a few years ago nobody would even care to mention on their resumes.

With a completely new version about to go mainstream (ECMAScript.next), nowadays JavaScript is not only the most popular programming language, but it has in fact become the main language for state-of-the-art web- ans software development projects.
